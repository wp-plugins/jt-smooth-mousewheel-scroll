== Description ==

You can adjust the scroll of the mouse wheel smoothly. (마우스 휠의 스크롤 이동을 부드럽게 조절할 수 있습니다.)

* Disable below IE8, Firefox
* IE8 이하, 파이어폭스에서는 적용되지 않습니다.
* Disable Mobile
* 모바일에서는 적용되지 않습니다.

== Installation ==

1. Download the plugin and extract it.
1. 플러그인을 다운로드 받습니다.
2. Upload the directory '/jt-smooth-mousewheel-scroll/' to the '/wp-content/plugins/' directory.
2. /wp-contnet/plugins/ 디렉토리 안에 /jt-smooth-mousewheel-scroll/ 디렉토리를 업로드 합니다.
3. Activate the plugin through the 'Plugins' menu in WordPress.
3. 워드프레스 플러그인 메뉴에서 플러그인을 활성화 합니다.
