== Description ==
마우스 휠의 스크롤 이동을 부드럽게 조절할 수 있습니다.

* IE8 이하, 파이어폭스에서는 적용되지 않습니다.
* 모바일에서는 적용되지 않습니다.

== Installation ==
1. 다운로드 받은 플러그인을 FTP를 통해 'Plugins' 폴더로 업로드 하거나, 
   워드프레스 관리자-> 플러그인 탭 에서 'jt-smooth-mousescroll-scroll' 검색 후 설치 
2. 'JT Smooth Mousewheel Scroll' 플러그인 활성화
